

# Tidio-Payroll

For more information about DDD and architecture please refer to docs directory

## Installation

```bash
make start
make migrations
make fixtures
```

## Testing

```bash
make phpunit
```

## Code-Style and Static Analysis

Check code style:
```bash
make cs
```

Fix code style
```bash
make cs-fix
```

Run static analysis:
```bash
make phpstan
```
